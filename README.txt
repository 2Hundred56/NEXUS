To play NEXUS on your computer:
Sorry, NEXUS has not reached playable level.

To develop NEXUS on your computer:
- Install your version of the SDL Developer Libraries from https://www.libsdl.org/download-2.0.php.
- Mac: Copy SDL2.framework to /Library/Frameworks
- Install NEXUS to your IDE of choice:
- Eclipse: Search for projects in NEXUS/dev/eclipse
- XCode: An XCode project is available in NEXUS/dev/xcode. WARNING: Make sure that test files aren't compiled.
